关于文件版本的分类

2023年02月02日 文档

即日起，小q将对未来即将制作的电子书文档之制作标准进行记录和存档，从而以更清晰的方式呈现版本差别及推荐对应阅读器（应用）。

👇差异 / 版本👉 顶配版 标准版 低配版
正文文本 ✓ ✓ ✓
内嵌字体 ✓ ✓ ✕
繁体版本¹ 看心情 ✓ ✕
售后服务² ✓ ✓ 看心情
全书视觉³ 高度还原 还原 简易
图片 最大限度高清 原图 原图
弹注 / 脚注 最大可能兼容 图片 文字

*注意：文件标准以最终制作发布版本为准。
*该标准会因为制作水平的提高和制作精细度的深入而不断改进，敬请留意。
1. 繁體版本：僅當原（譯）文為繁體中文時提供繁體中文版本，同時對簡體中文版文本進行適當本土化處理。
2. 售后服务：并没有售卖文件，所以提不提供最终解释权归我所有。恕不向转载及盗转网站提供售后服务。
3. 全书视觉：包括但不限于感想栏、制作信息、目录、小剧场等的原书装帧设计模仿。根据版本不同提供SVG矢量图插入、CSS代码绘制、PNG图片插入等不同形式和不同程度的复刻或模仿。
